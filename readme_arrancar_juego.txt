Para lanzar el proyecto:
	-Se requiere de la instalación de Angular y Xampp y del uso de IntelliJ.

1- Arrancar Xampp e iniciar el servicio MySql.

2- Abrir ambos archivos.

3- Arrancar el servidor de la parte del proyecto BackEnd.

4- En el proyecto FrontEnd incializar localhost.
	-Para ello abrimos la terminal, deberá mostarse la ruta donde esta el archivo guardado.
	En caso de que no aparezca, pulsamos ctrl + c, ahi deberia mostrarse.
	
	-Tecleamos 'ng serve' (sin las comillas) y apretamos intro.
	-Nos mostrará un enlace con localhost://4200, clicamos y abrirá el juego.

5- Para arrancar el juego debemos pulsar el boton de NUEVA PARTIDA y imtroduciremos nombre.
	
	Se procedera a escribir la linea de juego y se pulsará intro para realizar
	las comprobaciones.

	Tanto si el jugador acierta la palabra como si se queda sin oportunidades,
	el juego mostrará las letras acertadas y guardará los datos en la BBDD.

	Los datos se pueden comprobar en el boton VER RESULTADOS.
